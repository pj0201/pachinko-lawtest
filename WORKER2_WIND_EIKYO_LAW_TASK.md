# ワーカー2 タスク: 風営法（風適法）データ収集

**タスク名**: 取扱主任者講習テキスト用・風営法資料収集
**ステータス**: 実装中
**開始日**: 2025-10-20

---

## 📋 タスク概要

取扱主任者講習テキスト開発のため、**風営法（風俗営業等の規制及び業務の適正化等に関する法律）** に関連する法律資料を収集・整理する。

### 収集対象

1. **風営法 全文**
   - 現行法（最新版）
   - 重要条文の抽出

2. **関連法令**
   - 風営法施行令
   - 風営法施行規則
   - 厚生労働省通達

3. **改正情報**
   - 最新改正（令和X年X月）
   - 改正内容の要約

4. **参考資料**
   - 遊技機検定基準
   - 取扱主任者資格要件
   - 営業所管理基準

---

## 💾 保存場所について

### **👇 ワーカー2へのご相談内容**

収集した風営法データを、以下のいずれかの場所に保存したいと考えています：

#### **案1: `/resources/legal/` （推奨）**
```
patshinko-exam-app/
├── resources/
│   ├── legal/
│   │   ├── wind_eikyo_law/
│   │   │   ├── 風営法_全文.md
│   │   │   ├── 風営法_施行令.md
│   │   │   ├── 風営法_施行規則.md
│   │   │   ├── 最新改正_令和X年.md
│   │   │   └── 検定基準.md
│   │   └── reference/
│   └── ...
```
**利点**:
- アプリリソースとしてクリア
- 複数法律対応が容易
- 拡張性高い

---

#### **案2: `/docs/legal/` （ドキュメント型）**
```
patshinko-exam-app/
├── docs/
│   ├── legal/
│   │   ├── 風営法.md
│   │   ├── 施行令.md
│   │   └── 参考資料.md
│   └── ...
```
**利点**:
- ドキュメント整理が明確
- GitHub WIKI連携が容易

---

#### **案3: `/data/legal/` （データとして）**
```
patshinko-exam-app/
├── data/
│   ├── ocr_results_corrected.json
│   ├── legal/
│   │   ├── 風営法_全文.json
│   │   └── 風営法_条文_構造化.json
│   └── ...
```
**利点**:
- JSON構造化も可能
- 検索・参照が効率的
