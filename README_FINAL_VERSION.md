### 各フィールドの説明
- `problem_id`: 問題番号（1～1491）
- `theme_id`: テーマID
- `theme_name`: テーマ名
- `category`: カテゴリ（6大カテゴリ）
- `pattern_id`: パターンID（1～12）
- `pattern_name`: パターン名
- `difficulty`: 難易度（★～★★★★）
- `problem_type`: 問題タイプ（true_false）
- `format`: 形式（〇×）
- `problem_text`: 問題文（具体的な主張文）
- `correct_answer`: 正解（○または×）
- `explanation`: 解説
- `source_pdf`: ソースPDF番号
- `source_page`: ソースページ番号
