## ❌ まだカバーされていない重要な風営法条文と具体的内容

### 🔴 第1章：総則・許可の基礎（未カバー: 9条文）

#### **第3条** - 営業の許可（最も基本的な条文）
```
内容: 風俗営業を営もうとする者は、風俗営業の種別に応じて、営業所ごとに、
当該営業所の所在地を管轄する都道府県公安委員会の許可を受けなければならない。

重要性: 風営法の根幹をなす許可制度の基本規定
具体的項目:
- 営業所ごとの許可が必要
- 都道府県公安委員会が許可権者
- 種別に応じた許可
```

#### **第6条** - 暴力的不法行為等の罪（欠格事由の詳細）
```
内容: 国家公安委員会規則で定める暴力的不法行為その他の罪に当たる行為をした者で、
その行為の日から起算して5年を経過しない者は許可を受けられない。

具体的項目:
- 爆発物取締罰則違反
- 刑法の特定犯罪（殺人、傷害、逮捕監禁、強制性交等、わいせつ等）
- 組織的犯罪処罰法違反
- 麻薬・覚醒剤取締法違反
- 児童買春・児童ポルノ法違反
- 銃砲刀剣類所持等取締法違反
- 労働基準法・職業安定法違反
```

#### **第8条** - 許可証の交付・様式
```
内容: 公安委員会は、許可をしたときは、申請者に対し、許可証を交付する。

具体的項目:
- 許可証の様式
- 許可証の記載事項
- 許可証の提示義務
```

#### **第11条** - 届出事項の変更
```
内容: 風俗営業者は、届出事項に変更があったときは、遅滞なく、
その旨を公安委員会に届け出なければならない。

具体的項目:
- 氏名、住所の変更
- 法人の役員変更
- 営業所の名称変更
- 管理者の変更
- 届出期限と手続き
```

#### **第13条** - 地位の承継
```
内容: 風俗営業者である個人が死亡した場合、相続人がその地位を承継できる
（公安委員会の承認が必要）。

具体的項目:
- 相続による承継の要件
- 承認申請の期限（60日以内）
- 承認の基準
- 法人の合併・分割による承継
```

#### **第15条** - 騒音・振動の規制
```
内容: 風俗営業者は、営業所周辺において、政令で定めるところにより、
都道府県の条例で定める数値以上の騒音又は振動が生じないように営業を営まなければならない。

具体的項目:
- 騒音規制値（デシベル値）
- 振動規制値
- 測定方法
- 時間帯別の規制
- 違反時の罰則
```

#### **第16条** - 広告・宣伝の規制
```
内容: 風俗営業者は、その営業につき、営業所周辺における清浄な風俗環境を
害するおそれのある方法で広告又は宣伝をしてはならない。

具体的項目:
- 禁止される広告の内容
- 広告の掲示場所の制限
- わいせつな表現の禁止
- 違反時の罰則
```

#### **第17条** - 料金の表示
```
内容: 風俗営業者は、国家公安委員会規則で定めるところにより、
その営業に係る料金で国家公安委員会規則で定める種類のものを、
営業所において客に見やすいように表示しなければならない。

具体的項目:
- 表示すべき料金の種類
- 表示方法（見やすい場所、文字の大きさ等）
- 表示様式
- 変更時の手続き
```

#### **第19条** - 年少者立入禁止の表示
```
内容: 風俗営業者は、営業所の入口及び営業所内の見やすい場所に、
国家公安委員会規則で定める様式の標識を掲示しなければならない。

具体的項目:
- 標識の様式（大きさ、色、文言）
- 掲示場所（入口及び店内）
- 標識の文言（「18歳未満の者の立入りを禁ずる」等）
```

#### **第21条** - 帳簿の備付け
```
内容: 風俗営業者は、国家公安委員会規則で定めるところにより、
帳簿を備え、必要な事項を記載しなければならない。

具体的項目:
- 記載すべき事項（営業日時、売上、従業者等）
- 帳簿の保存期間（3年間）
- 電磁的記録での保存方法
- 検査時の提示義務
```

#### **第22条** - 苦情の処理
```
内容: 風俗営業者は、その営業に関し苦情の申出を受けたときは、
適切かつ迅速にこれを処理しなければならない。

具体的項目:
- 苦情処理の方法
- 苦情処理台帳の作成
- 苦情処理結果の記録
- 公安委員会への報告
```
