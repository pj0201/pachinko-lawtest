地方の方が厳しい規制を定めている場合、その条例が優先される。
この場合、営業者は5年ごとの届出義務を負う。

【思考レベル】
複数のルールが競合する場合の優先順位を判定。

### **パターン5: 時間経過による法的ステータス変化**

```
問題形式: 時系列判定
難易度: ★★★

【問題文】
営業者が営業許可を取得してから以下のような時間経過があった場合、
各時点での法的ステータスについて述べている。
正しいのはどれか？

① 許可取得直後～3年目：営業可能
   3年目：有効期限到達
   3年後（更新申請なし）：許可失効

② 許可取得直後～20年後：営業可能
   有効期限による失効がない ✅ 正解
   （ただし他の違反による取消は別）

③ 許可取得後1年ごとに更新届が必要

④ 毎年有効期限更新が必要

【解説】
営業許可は無期限だから：
- 取得から何年経っても有効
- 有効期限切れで失効しない
- 更新手続きが不要

（ただし、構造基準違反など他の理由で取消されることはあり、それは別問題）

【思考レベル】
時間軸上での「いつ何が起きるか」を理解する。
無期限 = 「何年経っても有効」という意味の理解を問う。
```

**このパターンが問う側面**: 時系列面 - 時間経過での法的変化

【思考レベル】
複数違反が同時に存在する場合、どれが最重大か判定する能力。

② 営業許可：無期限、型式検定：3年 ✅ 正解

### **パターン9: ルール背景・理由の理解**

```
問題形式: 理由理解
難易度: ★★★

【問題文】
営業許可が「無期限有効」と定められているのは、
以下のうちどれが最大の理由か？

① 営業者の経営効率を高めるため

② 営業許可は行政の適正な審査を経て許可基準を満たしており、
   その後、違反がない限り営業継続を認めるため ✅ 正解

③ 定期的な更新手続きによる行政負担を軽減するため

④ 営業者の不安を取り除くため

【解説】
許可制度の本質：
- 行政が初期審査で基準を満たしていると判定 → 許可付与
- 基準を満たしている限り、営業継続を認める
- 違反が生じれば、その時点で取消の手続きを進める

「無期限」は「基準を満たす限り営業継続を認める」という
法治主義の原則に基づく。

対比：
×「無期限で良い」（受動的）
○「基準を満たす限り営業継続を認める」（能動的な責任判定）

【思考レベル】
ルールの背景にある「目的」と「法治主義」の理解を問う。
単なる知識ではなく、法制度設計の思想を理解する能力。
```

**このパターンが問う側面**: ルール背景面 - なぜ無期限か

### **パターン10: 実務経験と法律の乖離（経験陥阱）**

```
問題形式: 経験陥阱判定
難易度: ★★★

【問題文】
長年営業してきた経営者が述べた：

「20年間、営業許可で営業してきているが、
営業許可は『更新が必要』だと思い込んでいた。
しかし、実際には更新申請をしなくても営業できている。
これはなぜか？」

この経営者の経験が矛盾しない理由は？

① 警察の目を逃れて不正営業している

② 営業許可は実は無期限有効で、
   更新申請が不要だから ✅ 正解

③ 営業許可が失効しているが、
   行政が見過ごしている

④ 都道府県の特例により更新不要

【解説】
実務 vs 法律の乖離：
- 経営者の思い込み：「営業許可は更新必要」
- 法律の実態：「営業許可は無期限、更新不要」
- 結果：思い込みが外れても、法律的に問題なし

この経験者は20年間、
「更新申請がない状態で営業継続」していることで、
実は法律が正しかったことを体験的に学んでいる。

【思考レベル】
「実務上そうしている」と「法律で定められている」の違いを認識する能力。
経験と知識の乖離を見極め、正しい法律理解に修正する能力。
```

**このパターンが問う側面**: 経験陥阱面 - 実務 vs 法律

風営法の改正により、営業許可制度について

③ 新ルールに従う義務はない

【思考レベル】

### **パターン12: 複合条件下での応用判定**

```
問題形式: 複合応用判定（最複雑）
難易度: ★★★★

【問題文】
営業者Cが以下の複雑なシナリオに直面した：

【状況】
- 営業許可：無期限有効（更新不要）
- 遊技機型式検定：あと1ヶ月で期限切れ
- 営業許可に基づく営業所設備改修工事の予定あり
- 遊技機の新台導入計画を立てている
- スタッフの取扱主任者資格更新も迫っている

営業者Cが最優先で対応すべき手続きは？

① 営業許可の更新（実は不要）
② 型式検定の更新申請
   （有効期限切れで営業不可になるため、最優先） ✅ 正解
③ 営業所の設備改修（基準維持のため）
④ 新台導入契約（ビジネス拡大のため）

理由：優先度判定
（1位）型式検定失効 > 営業許可失効なし
（2位）スタッフ資格更新 > 営業許可（無期限）
（3位）設備改修 > 新台導入

【解説】
営業継続に不可欠な要素の優先度：
1. **型式検定3年** ← 失効すると「その機械が営業不可」になる
2. **営業許可** ← 無期限なので更新不要、ただし失効すると「営業全体不可」
3. **取扱主任者資格** ← 失効すると「運営体制違反」
4. **営業所基準** ← 違反は取消リスク
5. **ビジネス拡大** ← 優先度最低

複合判定のテーマ：
営業許可が「無期限で心配不要」という理解が、
他の要件を軽視させてはいけない、
ということを学ぶ。

【思考レベル】
複数の時間制約・優先度を総合的に判定する能力。
各制度の相対的重要性を理解。
実務における「何を優先すべきか」の判断を問う。
```

**このパターンが問う側面**: 複合応用面 - 総合判定
