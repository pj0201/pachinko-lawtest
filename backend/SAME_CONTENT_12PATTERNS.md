# 同一内容を12パターンで記述 - パターン検証

**目的**: 同じ「営業許可の有効期限3年」という内容を、12パターンのそれぞれで問う

---

## 📝 基準となる内容

**内容**: 営業許可の有効期限は3年である（風営法第6条）

---

## パターン1: 基本知識の直接問い合わせ

```
営業許可の有効期限は何年か？

① 1年
② 2年
③ 3年 ✅
④ 5年
```

**特徴**: 知識があれば直接回答可能

---

## パターン2: 「必ず」「絶対」で例外見落とし（ひっかけ）

```
営業許可は、必ず有効期限から3年後に取り消される。

○ 正しい
× 誤り ✅

【正解】×

解説：有効期限は3年だが、期限切れ後に更新申請できる。
自動的に取り消されるわけではない（「必ず」という絶対表現が罠）
```

**特徴**: 「常に」「必ず」という絶対表現が例外を見落とさせる

---

## パターン3: 似た概念だが異なる用語

```
以下の2つの違いを述べよ：

A. 営業許可の有効期限：3年
B. 遊技機認定の有効期限：?

正しい対応はどれか？

① AとBは同じ3年
② Aは3年、Bは2年
③ Aは3年、Bは5年
④ Bは有効期限がない ✅（実は認定には別規定）
```

**特徴**: 同じく「許可」「認定」という似た概念だが、規制期間が異なる

---

## パターン4: 複数ルール間の優先順位

```
営業許可の有効期限が全国統一で3年と定められているが、
ある地方自治体では条例で「2年」と定めている場合、
営業者が従うべき有効期限は？

① 全国統一の3年
② 地方の2年 ✅
③ 営業者の選択で決定
④ 期間の平均（2.5年）
```

**特徴**: 複数のルール（国 vs 地方）が競合する場合の優先順位

---

## パターン5: 時間経過による法的ステータス変化

```
営業者が営業許可を受けた。その後の時間経過による
法的ステータスの変化について述べている。
正しいのはどれか？

① 許可取得直後～1年目：営業可能
   1年目～3年目：営業可能
   3年後：自動的に許可失効

② 許可取得直後～3年目：営業可能
   3年目：有効期限到達
   3年後（更新申請なし）：許可失効 ✅
   5年後：再申請可能

③ 許可取得後：永久に営業可能

④ 毎年更新申請が必要
```

**特徴**: 時間軸上での「いつ何が起きるか」を理解する必要

---

## パターン6: シナリオ判定（複雑な状況）

```
営業者Aが以下のシナリオに直面した：

【状況】
- 営業許可を受けてから2年11ヶ月経過
- 営業許可有効期限の更新申請期限は、
  「有効期限の30日前から申請可能」
- 営業者Aは更新申請を忘れていた
- 有効期限まであと10日

営業者Aが現在取るべき措置は？

① 10日待って、有効期限後に更新申請する
② 今すぐ更新申請する（期限内に間に合う） ✅
③ 営業許可失効後の営業しかできない
④ 元の許可で営業継続できる
```

**特徴**: 時間制限と手続きの複合的判定

---

## パターン7: 複数違反時の優先度判定

```
営業者が以下の2つの行為を同時に行った：

違反1: 営業許可の有効期限を3年と理解していたが、
      実際には2年だったため、更新手続きを忘れて1年延滞営業した

違反2: その1年間、営業所の構造基準を満たさず営業した

より重大な違反（優先して処分される）は？

① 違反1（有効期限の誤解）
② 違反2（構造基準違反） ✅
③ 両者同等
④ 違反1が優先される
```

**特徴**: 複数違反が同時に存在する場合、どれが最重大か

---

## パターン8: 数値・統計データの正確性

```
営業許可の有効期限に関する以下の数値について、
正しいのはどれか？

① 有効期限：2年
② 有効期限：3年 ✅
③ 有効期限：4年
④ 有効期限：5年

追加問題：その3年のうち、更新申請が可能な期間は？

① 0～30日前
② 30日～0日前（有効期限30日前から申請可能） ✅
③ 有効期限後30日以内
④ 更新申請期限の定めがない
```

**特徴**: 正確な数値と日数の把握が必須

---

## パターン9: ルール背景・理由の理解

```
営業許可の有効期限が「3年」と定められているのは、
以下のうちどれが最大の理由か？

① 営業者の経営効率を高めるため
② 一度許可を得たら永久にしないと、
   営業者の負担が大きくなるため
③ 定期的に営業者の資質・営業所の状態を確認し、
   適切な風俗環境を維持するため ✅
④ 政府の税収を増やすため（更新手数料）
```

**特徴**: ルールの背景にある「目的」の理解が問われる

---

## パターン10: 実務経験と法律の乖離（経験陥阱）

```
長年営業してきた経営者が述べた：

「20年間、営業許可で営業してきている。
許可が有効期限3年といっても、
実際には継続営業が認められているから、
更新手続きをしなくても営業できるはずだ」

この発言は法的に？

① 完全に正しい（実務が優先される）
② 誤り。3年ごとに更新申請が必須で、
   手続きを怠ると許可失効となる ✅
③ 経営者の判断に委ねられる
④ 都道府県によって異なる
```

**特徴**: 「実務上そうしている」と「法律で定められている」の違い

---

## パターン11: 最新法令改正への対応

```
風営法の改正により、営業許可の有効期限の
取り扱いについて新しいルールが追加された場合
（仮想シナリオ）、既に3年の許可を持つ営業者の対応は？

① 新ルールは施行日以降の許可に適用され、
   既存許可は3年のまま（経過措置あり） ✅

② 既存許可も直ちに新ルール適用される

③ 新ルールに従う義務はない

④ 改正法は遡及しない（完全に新ルール適用なし）

追加問題：改正前に取得した許可が有効期限を迎える場合、
どのルールで更新するか？

→ 施行日を境に異なるルール適用の可能性
```

**特徴**: 改正法の「遡及適用」と「経過措置」を理解する必要

---

## パターン12: 複合条件下での応用判定

```
営業者Bが以下の複雑なシナリオに直面した：

【状況】
- 営業許可有効期限：あと1ヶ月
- 営業許可更新申請手数料：33,600円（予算が足りない）
- 同時期に遊技機の大型更新を計画している
- 営業所の構造改修工事を進めている
- スタッフの遊技機取扱主任者資格が更新時期

営業者Bが最優先で対応すべき手続きは？

① 営業許可の更新申請
   （有効期限失効で営業不可になるため、最優先） ✅

② 遊技機の更新（営業をより良くするため）

③ 営業所の改修（構造基準を満たすため）

④ スタッフの資格更新（人的基準を満たすため）

理由：営業許可失効 > その他すべての課題
（許可がなければ、どの事業もできない）

追加判定：もし営業許可が失効した場合、
その後の対応は？

→ 5年間は再申請不可（許可取消に準ずる扱い）
   ただし、実務上は異議申し立て等の救済手段あり
```

**特徴**: 複数の時間制約・予算制約・人的資源の複合的判定

## ✅ 結論

### 12パターンの妥当性確認

**✅ 12パターンで十分**

理由：
1. 同一内容で12パターンすべてが独立して成立
2. 各パターンが異なる「思考スキル」を問う
3. 「知識 → 応用 → 複合判定」に段階化している
4. 難易度分布が適切（★から★★★★へ）

### 推奨パターン分布（修正版）

```
パターン1-8（★～★★）：  35%（基礎理解）
  - 基本知識、ひっかけ、用語、優先順位、数値

パターン9-12（★★★～★★★★）： 65%（応用理解）
  - 時系列、シナリオ、複合、経験陥阱、改正対応、総合判定
```

これで、**仕様書は完成**できます。
