# 遊技機試験 - 12パターンサンプル問題

**作成日**: 2025-10-22
**目的**: 12パターンの妥当性検証と品質シミュレーション

## 📝 パターン1: 基本知識の直接問い合わせ

**特徴**: 法律条文・ルールの直接確認

### サンプル問題 1-1
```
パチンコ店の営業時間の原則は、午前10時から午後11時までである。
○ 正しい
× 誤り
```

**解説**: 風営法第8条により、営業時間の原則は10時～23時。

### サンプル問題 1-2
```
営業許可を受けるには、都道府県公安委員会に申請書を提出しなければならない。
○ 正しい
× 誤り
```

**解説**: 風営法第3条による基本的な手続き。

### サンプル問題 1-3
```
遊技機の型式検定は、都道府県公安委員会が実施する。
○ 正しい
× 誤り
```

**解説**: 風営法第20条により、認定及び型式検定は公安委員会。

## 📝 パターン2: 「必ず」「絶対」で例外見落とし（ひっかけ）

**特徴**: 絶対表現を含めて、例外を見落とさせる

### サンプル問題 2-1
```
営業許可を受けた後、営業所の構造や設備を変更する場合は、
必ず公安委員会に届け出なければならない。
○ 正しい
× 誤り
```

**解説**: 「軽微な変更」であれば届け出不要（風営法第9条第2項）。
引っかけポイント：「必ず」という絶対表現で例外を見落とさせる

### サンプル問題 2-2
```
遊技機の不正行為は、絶対に報告義務がある。
○ 正しい
× 誤り
```

**解説**: 不正を発見した場合の報告義務はあるが、「軽微な不正」や「疑わしい段階」の報告義務との厳密な区別あり。

### サンプル問題 2-3
```
営業者は、常に遊技機の日常管理を自ら行わなければならない。
○ 正しい
× 誤り
```

**解説**: 業務委託が可能な場合がある（遊技機取扱主任者による委託など）。

## 📝 パターン3: 似た概念だが異なる用語

**特徴**: 概念の違いを正確に理解

### サンプル問題 3-1
```
「営業許可」と「営業認可」は同じ意味である。
○ 正しい
× 誤り
```

**解説**: 異なる。「許可」は営業開始、「認可」は特定の行為（中古台の設置等）。

### サンプル問題 3-2
```
「遊技機の型式検定」と「遊技機の認定」は異なるものである。
○ 正しい
× 誤り
```

**解説**: 正しい。
- 型式検定：新しい機種が基準に適合するか確認
- 認定：検定済み機種が実際の営業所の設備で使用可能か確認

### サンプル問題 3-3
```
「営業停止処分」と「営業許可の取消」は同じ効果である。
○ 正しい
× 誤り
```

**解説**: 異なる。停止は一時的、取消は永続的。

## 📝 パターン5: 時間経過による法的ステータス変化

営業を開始しなかった場合、許可は失効する。

その期間は最大何日か？
① 30日
② 60日

## 📝 パターン6: シナリオ判定（複雑な状況）

**特徴**: 複数の条件を含む複雑なシナリオの判定

### サンプル問題 6-1
```
営業者は遊技機を新たに設置しようとしている。
- 新品パチンコ機
- 型式検定済み
- 営業所構造は基準を満たす
- ただし、営業許可申請時と営業所の住所が変更している

この場合、営業者がすべき手続きは？
① 何もしなくても設置可能
② 住所変更届を提出すれば設置可能
③ 変更許可申請が必要
④ 営業許可を取り直す必要がある
```

**解説**: ③ 変更許可申請が必要

### サンプル問題 6-2
```
営業所で以下の状況が発生した：
- 従業員が遊技機を分解し、内部の基板を改造した
- その改造は検定に不合格のものだった
- 改造行為を発見したのは遊技機取扱主任者

遊技機取扱主任者の義務は？
① 公安委員会に報告し、遊技機を撤去する
② 営業者に報告し、営業者の判断に委ねる
③ 何もしなくても法的問題はない
④ 当該遊技機の使用を禁止するが、報告は不要
```

**解説**: ① 公安委員会に報告し、撤去が必要

## 📝 パターン10: 実務経験と法律の乖離（経験陥阱）

**特徴**: 実務で「そうなっている」と「法律で定められている」の違い

### サンプル問題 10-1
```
長年パチンコ店の経営をしている営業者が述べた：
「うちの店では営業許可以来、毎日営業しているから、
営業を開始する期限の問題は関係ない」

この発言は法的に？
① 完全に正しい
② 後日許可を取り消されるリスクがある
③ 営業を開始した時点で問題は解決している
④ 営業許可の有効期間が延長される
```

**解説**: ③ 営業を開始した時点で問題は解決している
（ただし、サンプル問題では「許可から6ヶ月以内に営業開始しなかった場合」という期限を把握していることが重要）

### サンプル問題 10-2
```
営業者が述べた：
「これまで景品の額を決めるときは、営業所の経営状況に応じて
柔軟に対応してきた。法律では上限額があるが、
営業所ごとに実情に合わせて対応しても問題ない」

この発言は？
① 経営の自由度として許容される
② 法律違反である
③ 地方自治体の許可があれば可能
④ 景品交換業者の判断に委ねられる
```

**解説**: ② 法律違反である。上限額は絶対（法定基準）。

## 📝 パターン12: 複合条件下での応用判定

**特徴**: 複数の条件が組み合わさった応用的な判定

### サンプル問題 12-1
```
以下の状況で営業者が取るべき措置は？

状況：
- 営業許可有効期限があと1ヶ月で切れる
- 新品パチンコ機を3台、新たに設置予定
- 営業所の構造改修を計画している
- 従業員の一部が遊技機取扱主任者資格更新時期

営業者が優先すべき手続きは？
① まず営業許可の更新申請
② 遊技機設置の認可申請
③ 従業員の資格更新
④ 複数を同時進行する
```

**解説**: ① 営業許可の更新申請が最優先（有効期限切れで営業不可）

### サンプル問題 12-2
```
営業所で以下が同時に発生した：
1. 営業許可取消を受けた（5年間再申請不可）
2. 遊技機の不正が発見された
3. 従業員が違反行為を行った

この営業所の法的ステータスは？
① 営業できなくなり、その後の復帰は困難
② 1年経てば営業再開可能
③ 条件を満たせば別の場所で営業可能
④ 遊技機の入替えで営業再開可能
```

**解説**: ① 営業できなくなり、5年間の再申請不可期間がある。
その後の復帰も多くの課題がある。
