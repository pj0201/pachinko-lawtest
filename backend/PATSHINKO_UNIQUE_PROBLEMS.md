# 遊技機試験特有の3つの問題パターン

**作成日**: 2025-10-22
**重要**: 運転免許試験には存在しない、遊技機試験固有の要素

---

## 🎮 特有パターン1: 遊技機の技術仕様と法律の複合理解

**特徴**: 遊技機の技術的構造と、それに対する法律規制の複合的理解

### サンプル問題 1-1
```
遊技機の「主基板」について述べている。正しいのはどれか？

① 主基板の改造は、営業所の管理者判断で軽微な改造なら許容される
② 主基板には暗号化やセキュリティ機能が要求され、
   改造や分解は厳禁であり、発見された場合は型式検定が取り消される
③ 主基板の性能には制限がなく、営業店の希望に応じてカスタマイズできる
④ 主基板の情報はメーカーの営業秘密であり、営業者は知る必要がない

正解: ②
```

**解説**: 遊技機取扱主任者は、遊技機の「技術的な機能」と「法的な規制」の両方を理解する必要がある。主基板はセキュリティの中核であり、改造は重大な法違反。

### サンプル問題 1-2
```
ホールコン（ホール管理用コンピュータ）について正しいのはどれか？

① ホールコンは遊技機に接続できる附属機器で、
   営業者が自由にカスタマイズして売上管理に利用できる
② ホールコンは遊技機と接続される場合、検定基準を満たす必要があり、
   不正な接続や改造は遊技機の型式検定取消につながる
③ ホールコンは営業管理ツールであり、法律の規制対象外である
④ ホールコンの接続は、個々の営業店の経営判断に委ねられている

正解: ②
```

**解説**: 遊技機の附属機器であるホールコンは、遊技機の検定対象に含まれる。不正接続は重大な違反。

### サンプル問題 1-3
```
新台遊技機を営業所に設置する際、遊技機取扱主任者が
確認すべき技術仕様は以下のうちどれか？（複数選択）

① 基板の型式と検定番号が、納品書と一致するか
② 外部端子板の接続方法が、認定仕様と一致しているか
③ チップ（回胴式遊技機の「リール」部分）の回転速度が、
   検定基準の範囲内か
④ ドア開閉機構が、検定仕様と異なっていないか

正解: ①②③④ すべて
```

**解説**: 遊技機は「外見」だけでなく、内部の技術仕様も検定対象。取扱主任者は技術的な検査能力が要求される。

## 🎁 特有パターン2: 景品交換所の法的地位と複数法律の相互関係

**特徴**: 景品交換業務が複数の法律で規制される複雑性

### サンプル問題 2-1
```
景品交換所について述べている。正しいのはどれか？

① 景品交換所は風営法の直接規制の対象外であり、
   交換所が景品額を自由に設定できる
② 景品交換所は「特例風俗営業」として風営法で規制され、
   また景品表示法による上限額制限も受ける
③ 景品交換所は営業者が自由に設置でき、運営方法に法的制限はない
④ 景品交換所の運営は、景品提供企業（営業店）の完全な裁量で決定される

正解: ②
```

**解説**: 景品交換業務は「風営法」「景品表示法」「消費者庁ガイドライン」など複数の法律が相互に規制する。1つの法律だけでは不十分。

### サンプル問題 2-2
```
営業店が景品交換所を利用する際、遵守すべき条件は以下のうちどれか？
（複数選択）

① 交換所との契約内容が風営法の基準を満たしているか確認
② 交換所の運営者が風営法上の「特例風俗営業」の許可を持っているか確認
③ 交換所で設定される交換比率が、景品表示法の上限額内か確認
④ 交換所の帳簿記載が、営業店の帳簿と照合可能か確認

正解: ①②③④ すべて
```

**解説**: 営業店は景品交換所との関係においても、複数の法律遵守責任がある。

### サンプル問題 2-3
```
景品交換所の運営者が以下を行った場合、
最も重大な法的問題が生じるのはどれか？

① 交換比率を「景品表示法の上限額」ちょうどに設定した
② 交換所の運営許可（特例風俗営業）を更新せず、
   許可の失効後も営業を継続した
③ 交換所の帳簿記載が不十分だった
④ 交換所の場所が学校から300メートル以内だった

正解: ②
```

**解説**: 許可失効後の営業は最重大。風営法違反として処分の対象。

## 🔄 特有パターン3: 新台と中古台の複雑な規制体系

**特徴**: 遊技機の「ライフサイクル」に応じた異なる規制が存在

### サンプル問題 3-1
```
新品遊技機と中古遊技機の「型式検定」について述べている。
正しいのはどれか？

① 新品遊技機は型式検定を受ける必要があるが、
   中古遊技機は既に検定済みなため、改めて検定は不要
② 新品遊技機は型式検定を受け、中古遊技機は「認定」を受ける
   という異なるプロセスが必要
③ 型式検定と認定に法的違いはなく、単に呼び方が異なるだけ
④ 中古遊技機は、いかなる場合も営業所に設置できない

正解: ②
```

**解説**: **遊技機試験特有の複雑さ**
- 新品 → 「型式検定」（新しい機種が基準を満たすか）
- 中古 → 「認定」（既検定機種が特定の営業所で使用可能か）

### サンプル問題 3-2
```
中古遊技機を営業所に設置する際の手続きについて述べている。
正しいのはどれか？

① 中古遊技機は已に検定済みなので、届出のみで設置可能
② 中古遊技機を設置するには、以下のプロセスが必要：
   1) 販売業者から遊技機を購入
   2) 保守管理業者（または販売業者）による点検・確認
   3) 認定申請を公安委員会に提出
   4) 認定を受けた後に設置
③ 中古遊技機の設置に許可は不要。営業者の判断で自由に設置可能
④ 中古遊技機は、新たに型式検定を受ける必要がある

正解: ②
```

**解説**: **遊技機試験特有の多段階プロセス**

中古遊技機は：
- 単なる「購入」ではなく
- 複数の確認・申請プロセス
- 販売業者・保守管理業者・公安委員会の関係
が複雑に絡み合う。

### サンプル問題 3-3
```
以下の状況について述べている。正しい対応はどれか？

【状況】
- 営業店A が、同じメーカーの遊技機を3台購入予定
- うち2台は新品、1台は別の営業店からの中古
- 営業店B が営業許可の取消を受け、その遊技機を譲受

営業店A が取るべき措置は以下のうちどれか？

① 新品2台：型式検定を確認後、認定申請で対応
   中古1台：営業店B から譲受できないため、購入不可

② 新品2台：型式検定済み機種なら、認定申請で対応
   中古1台：営業店B の許可取消後なため、確認が必要

③ すべて同じプロセス：型式検定を確認後、認定申請

④ 3台とも：営業店A の営業許可さえあれば、制限なし

正解: ②
```

**解説**: **遊技機試験特有の複雑性の最高峰**

- 新品と中古で異なるプロセス
- 譲受元の営業店のステータス（許可取消中）が影響
- 販売業者・保守管理業者による確認が必須
- 複数の時点での確認が必要

## 📊 遊技機試験特有パターンの特徴

### 運転免許試験との決定的な違い

| 要素 | 運転免許試験 | 遊技機試験 |
|------|-----------|---------|
| **規制対象** | 運転者の行動 | 機器・システム + 営業形態 |
| **技術知識** | 不要 | 必須（遊技機の仕様） |
| **複数法律** | 少ない | 多い（風営法+景品表示法等） |
| **ライフサイクル** | 車の「新古」区別少ない | 遊技機の「新台/中古」で大きく異なる |
| **事業者との関係** | 一者（ドライバー） | 複数者（営業者、販売業者、交換所等） |

### 遊技機試験特有パターンの3つ

```
【特有パターンA】遊技機の技術仕様と法律の複合理解
  - 基板、ホールコン、チップ等の技術を理解した上での法律判定

【特有パターンB】景品交換所の複数法律相互関係
  - 1つの事業（景品交換）が複数の法律に規制される複雑性

【特有パターンC】新台と中古台の異なる規制体系
  - 同じ遊技機でも、ライフサイクルステージで規制が異なる
```
