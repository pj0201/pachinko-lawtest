# デモンストレーション：同一テーマの12パターン問題

**テーマ**: 「遊技機の型式検定は3年有効」
**目的**: 1つのテーマがいかに12の異なる思考レベルで出題されるかを実証
**表示形式**: 1画面版（A4縦向きに対応）

## 📝 12パターン問題セット

### **パターン1: 基本知識【知識確認】★**
```
遊技機の型式検定の有効期限は何年か？

① 1年 ② 2年 ③ 3年 ④ 5年

【正解】③ 3年
【解説】遊技機の型式検定は3年有効。期限到達時に更新申請が必要。
```

### **パターン2: ひっかけ【絶対表現への注意】★★**
```
遊技機の型式検定は必ず3年ごとに更新申請が必須である。

○ 正しい × 誤り

【正解】× 誤り（「必ず」が罠。有効期限内なら申請不要）
【解説】3年有効だが、期限内に申請すれば継続営業可能。
```

### **パターン3: 用語比較【概念の区別】★★**
```
営業許可と型式検定の有効期限の違いは？

A. 営業許可：無期限
B. 型式検定：3年

正しい対応は？ ① 同じ ② 異なる ③ 反対

【正解】② 異なる（営業許可=無期限、型式検定=3年）
【解説】営業許可は無期限で更新不要。型式検定は3年で更新必須。
```

### **パターン4: 優先順位【法律体系の理解】★★**
```
営業許可は無期限だが、遊技機の型式検定は3年有効という
2つのルールがある場合、営業者が最優先で対応すべきは？

① 営業許可の更新
② 型式検定の更新申請 ← より重要

【正解】② 型式検定（期限超過で営業不可になる）
【解説】営業許可は期限なし。型式検定期限切れが営業停止の直接原因。
```

### **パターン5: 時系列理解【時間経過による変化】★★★**
```
営業者が新台を設置して以下の時間経過があった場合、
法的ステータスの変化について述べている。正しいのは？

① 設置当初3年：営業可能
   3年後：自動失効

② 設置当初3年：営業可能
   3年時点：更新申請で継続可能 ✅
   更新なし：営業不可

【正解】②（更新申請により継続営業可能）
【解説】3年経過で自動失効ではなく、更新申請で継続可能。
```

### **パターン6: シナリオ判定【実務状況判定】★★★**
```
営業店Xが遊技機設置から2年11ヶ月時点で以下に気づいた：
「型式検定の有効期限がもうすぐ切れる。
更新申請期限は期限30日前から」

今すぐ申請すべき？ ① はい ② いいえ

【正解】① はい（期限の30日前から申請可能。今が申請時期）
【解説】2年11ヶ月 = あと1ヶ月で期限。30日前条件を満たす。
```

### **パターン7: 複合違反【優先度判定】★★★**
```
営業店が2つの違反を同時に行った：

違反1: 型式検定期限切れで旧機械を放置
違反2: 営業許可なし状態で営業

どちらが重大？ ① 違反1 ② 違反2 ③ 同等

【正解】② 違反2（営業許可なし = 営業全体が違反）
【解説】型式検定期限切れ < 営業許可なし（全体的問題）
```

### **パターン8: 数値正確性【正確な知識確認】★**
```
型式検定の有効期限について正しい組み合わせは？

A. 有効期限：3年
B. 更新申請開始：期限30日前

① AのみOK ② BのみOK ③ A・B両方OK ④ 両方ダメ

【正解】③ A・B両方OK（両方正確）
【解説】3年が正確な期限。30日前申請が標準規定。
```

### **パターン9: 理由理解【法制度設計の理解】★★★**
```
型式検定がなぜ「無期限」ではなく「3年有効」なのか？
最大の理由は？

① コスト削減
② 遊技機の技術進化に対応するため ✅
③ 営業者の負担軽減
④ 政府の収入増

【正解】②（新技術対応、不正検出のため定期的な検査が必須）
【解説】3年ごとの更新で、新しい不正改造技術に対応。
```

### **パターン10: 経験陥阱【実務と法律の乖離】★★★**
```
経験者が述べた：「10年営業してるが、
型式検定3年を知らずに、同じ機械を使い続けていた」

この現象は何を示す？

① 警察の目を逃れた ② 規定が不正確だった
③ 営業許可と誤解していた ④ 不違反だった

【正解】③ 営業許可（無期限）と型式検定（3年）を混同
【解説】営業許可は無期限だが、機械側は3年更新が別途必須。
```

### **パターン11: 改正対応【法改正への対応】★★★**
```
法改正で「型式検定を3年から5年に延長する」と決定。
既に3年許可を持つ営業者の対応は？

① 施行日以降、既存許可も5年に変更される
② 既存許可は3年のまま。新規は5年 ✅

【正解】②（経過措置で既存許可は保護される）
【解説】改正法の遡及適用は原則しない。既得権を保護。
```

### **パターン12: 複合応用【総合判定】★★★★**
```
営業店Yが直面する複合シナリオ：

・型式検定期限：あと20日
・営業許可：無期限で問題なし
・新台導入計画あり
・スタッフ資格更新も迫る

最優先対応は？

① 営業許可の更新申請
② 型式検定の更新申請 ← 最優先
③ 新台導入
④ スタッフ資格更新

【正解】②（20日で期限切れ → 営業不可になる最大リスク）
【解説】営業許可は無期限なので更新不要。機械側の期限が優先。
```
