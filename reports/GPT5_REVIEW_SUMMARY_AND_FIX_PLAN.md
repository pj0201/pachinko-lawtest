## 🔍 共通問題点（頻度順）

### 1. 法的根拠の抽象性（5/6回で指摘）⭐⭐⭐⭐⭐

**問題**:
- 「施行規則」「技術規格」だけで条文番号が明記されていない
- 「民法」「リサイクル法」のみで具体的な条項がない

**具体例**:
```
❌ 法的根拠: 施行規則
❌ 法的根拠: 技術規格
❌ 法的根拠: 民法

✅ 法的根拠: 施行規則第36条第1項
✅ 法的根拠: 技術規格第5条第2項
✅ 法的根拠: 民法第96条
```

**影響範囲**: 230問（セルフレビューで検出済み）

---

### 2. 抽象的表現（5/6回で指摘）⭐⭐⭐⭐⭐

**問題**:
- 「適切な」「所定の」「一定の」などの曖昧な表現
- 受験者が具体的な基準を理解できない

**具体例**:
```
❌ 遊技機の設置には一定の間隔が必要である。
❌ 適切な手続きが必要である。
❌ 所定の期間内に届け出る。

✅ 遊技機の設置には都道府県条例で定める間隔が必要である。
✅ 風営法第7条に定める手続きが必要である。
✅ 60日以内に届け出る。
```

**影響範囲**: 101問（セルフレビューで検出済み）

---

### 3. 問題文と解説の不一致（4/6回で指摘）⭐⭐⭐⭐

**問題**:
- 問題文で問われている内容と解説が異なる
- 問題文が誤解を招く表現

**具体例**:
```
❌ 問題文: 遊技機の性能は技術規格に適合する。
   解説: 誤りです。正しくは、遊技機の性能は技術規格に適合する。
   → 問題文と解説が矛盾

✅ 問題文: 遊技機の性能は技術規格に適合しなければならない。
   解説: 正しい。風営法第20条により、遊技機の性能は技術規格に適合する必要があります。
```

**影響範囲**: 要全体精査

---

### 4. 誤った日本語表現「当該通りです」（3/6回で指摘）⭐⭐⭐

**問題**:
- 「当該通りです」という誤った日本語表現
- 正しくは「その通りです」「正しいです」

**具体例**:
```
❌ 解説: 当該通りです。中古機流通には流通制御端末が必要。
❌ 解説: 当該通りです。すべての遊技機には固有の製造番号が付与される。

✅ 解説: 正しいです。中古機流通には流通制御端末が必要。
✅ 解説: その通りです。すべての遊技機には固有の製造番号が付与される。
```

**影響範囲**: 全問を検索して修正必要

---

### 5. 指示語の使用（2/6回で指摘）⭐⭐

**問題**:
- 「これ」「それ」「その」などの指示語が残存
- 何を指すか不明確

**備考**: セルフレビュー + 自動修正で大半は削除済み（96件→4件、-95.8%）

## 🎯 修正方針（優先度順）

### 優先度：最高（即座に修正）

#### 1. 誤った日本語表現の修正
- **対象**: 全634問
- **方法**: 全文検索で「当該通りです」「当該通り」を「正しいです」「その通りです」に置換
- **実施**: 自動置換 + 手動確認

#### 2. 問題文と解説の不一致の修正
- **対象**: 全634問を精査
- **方法**:
  1. 問題文が「正しい」内容を述べている場合、解説で「誤りです」としていないか確認
  2. 問題文が「誤り」内容を述べている場合、解説で「正しいです」としていないか確認
- **実施**: Opusモデルで1問ずつ精査

---

### 優先度：高（早急に修正）

#### 3. 法的根拠の具体化
- **対象**: 230問（法的根拠なし または 抽象的）
- **方法**:
  - 「施行規則」→「施行規則第X条」
  - 「技術規格」→「技術規格第X条」
  - 「民法」→「民法第X条」
- **実施**: 法令データベースとの照合 + 専門家レビュー推奨

#### 4. 抽象的表現の具体化
- **対象**: 101問
- **方法**:
  - 「一定の」→「都道府県条例で定める」または具体的な数値
  - 「適切な」→具体的な法令条文
  - 「所定の」→具体的な期間・基準
- **実施**: Opusモデルで具体化案を生成

---

### 優先度：中（随時対応）

#### 5. 短文の拡張
- **対象**: 180問（20字未満）
- **方法**: 具体的な状況説明、数値、期限を追加
- **目標**: 30-70字
- **実施**: 意味変更リスクを考慮して慎重に対応
