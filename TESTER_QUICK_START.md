# テスター向けクイックスタートガイド

## 🎯 アルファ版テストへようこそ！

このアプリは、遊技機取扱主任者試験の学習をサポートするWebアプリケーションです。
アルファ版テストにご協力いただき、ありがとうございます。

## 📱 登録方法

### ステップ1: 招待URLにアクセス

配布された招待URL（例：`https://xxx.railway.app/invite/xxxxxx`）をクリックしてください。

### ステップ2: ユーザー名を入力

- ユーザー名を入力してください（例：テスター001、田中太郎など）
- 任意の名前でOKです

### ステップ3: 登録完了

「登録して始める」ボタンをクリックすると、自動的にホーム画面に移動します。

## 🎮 アプリの使い方

### ホーム画面

- **模擬試験を始める**: 新しい模擬試験を開始
- **成績を見る**: これまでの試験結果を確認

### 模擬試験

1. **難易度を選択**
   - ★（基礎）: 基本的な問題
   - ★★（標準）: 中級レベルの問題
   - ★★★（応用）: 難しい問題

2. **問題数を選択**
   - 10問、20問、30問、40問、50問から選択

3. **問題に回答**
   - 各問題を読んで、○（正しい）または×（誤り）を選択
   - 回答後、正解と解説が表示されます

4. **結果確認**
   - すべての問題に回答すると、総合結果が表示されます
   - 正答率、合格/不合格判定を確認できます

### 成績履歴

- 過去に受験したすべての模擬試験の結果を確認できます
- 各試験をクリックすると、詳細な内容を確認できます

## ⚠️ 重要事項

### デバイスの制限

- **1つの招待URL = 1台のデバイス**
- 配布された招待URLは、あなたのデバイス（スマホまたはPC）専用です
- 他の人とURLを共有しないでください

### ブラウザの推奨

- **推奨ブラウザ**: Chrome, Safari, Edge, Firefox（最新版）
- プライベートブラウジング（シークレットモード）での使用は避けてください

### データの保存

- 試験結果はブラウザに保存されます
- ブラウザのデータを削除すると、履歴も削除されます

## 🐛 バグ・問題を見つけた場合

以下の情報を記録してください：

1. **エラーメッセージ**: 画面に表示されたエラーの内容
2. **操作手順**: エラーが発生する直前に行った操作
3. **スクリーンショット**: 可能であれば画面キャプチャ
4. **デバイス情報**:
   - スマートフォン/PC
   - OS（iOS, Android, Windows, macOSなど）
   - ブラウザ（Chrome, Safariなど）

## 🎓 学習のコツ

1. **基礎から始める**: まずは★（基礎）レベルで慣れましょう
2. **少しずつ進める**: 10問ずつ解いて、理解を深めましょう
3. **解説を読む**: 間違えた問題は必ず解説を確認しましょう
4. **繰り返し練習**: 同じレベルを何度も解いて定着させましょう

## ❓ よくある質問

### Q1: ログイン情報を忘れました

A: 招待URLから再度アクセスすれば、自動的にログインできます（同じデバイスの場合）

### Q2: 異なるデバイスでも使いたい

A: 1つの招待URLは1台のデバイス専用です。追加のデバイスが必要な場合はお知らせください。

### Q3: 試験結果を削除できますか？

A: 現在のアルファ版では、試験結果の削除機能はありません。次のバージョンで追加予定です。

### Q4: 問題に間違いを見つけました

A: 問題の内容に関する指摘も大歓迎です！問題番号と具体的な内容をお知らせください。
