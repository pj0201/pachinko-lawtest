# 法律系正誤式問題リサーチ結果
作成日: 2025-10-23
目的: 遊技機取扱主任者試験問題の品質向上のため

## 📚 調査対象試験

### 1. 宅地建物取引士（宅建士）試験
**試験形式**:
- 50問・四肢択一マークシート
- 正誤問題が40問以上（最多）
- 個数問題：6問程度
- 組合せ問題：1-2問

**問題の特徴**:
- 単純正誤問題が中心
- 法令の正確な理解を問う
- 実務的な知識の確認

**合格ライン**: 31-37点（年度により変動）

**参考URL**: https://www.agaroot.jp/takken/column/question-format/

---

### 2. ファイナンシャルプランナー（FP）3級試験
**試験形式**:
- 学科60問（マークシート）
- 問1-30: ○×問題（2択）
- 問31-60: 3択問題

**問題配分**（各分野均等）:
1. ライフプランニングと資金計画（各分野5問）
2. リスク管理
3. 金融資産運用
4. タックスプランニング
5. 不動産
6. 相続・事業承継

**合格ライン**: 60%（36点/60点）

**2024年からの変更**: CBT方式（通年実施）

**参考URL**: https://www.jafp.or.jp/exam/mohan/

---

### 3. ビジネス実務法務検定 3級
**試験形式**:
- IBT（インターネット試験）
- 組み合わせ問題：複数の記述（ア～オ）の正誤組み合わせ

**問題の特徴**:
- 民法、会社法、労働法などビジネス法務全般
- 実務的な法律知識を問う
- 簿記3級レベルの難易度

**2024年からの変更**: CBT化により過去問非公開

**参考URL**: https://kentei.tokyo-cci.or.jp/houmu/

---

### 4. 社会保険労務士試験
**試験形式の特徴**:
- ほぼ100%が正誤問題
- 一問一答形式（肢別）
- 5肢択一が一般的だが、各肢が独立した正誤判断

**問題の特徴**:
- 法令の条文知識が中心
- 数値（期間、金額）の正確性を問う
- 手続きの流れ・順序を問う

## 🎯 問題作成のベストプラクティス（全試験共通）

### 1. 1問1論点の原則
**NG例**:
```
遊技機の設置には届出が必要であり、かつ型式検定に合格していなければならず、
さらに営業許可を取得している必要がある。
```
→ 3つの論点が混在

**OK例**:
```
遊技機を営業所に新たに設置する場合、公安委員会への事前届出が必要である。
```
→ 1つの論点のみ

---

### 2. 具体性と明確性
**NG例**:
```
これには届出が必要である。
```
→ 「これ」が何を指すか不明

**OK例**:
```
遊技機の型式を変更する場合には、公安委員会への届出が必要である。
```
→ 主語・対象・手続きが明確

---

### 3. 簡潔な表現（30-60文字が目安）
**長すぎる例（NG）**:
```
風俗営業等の規制及び業務の適正化等に関する法律第4条第1項第5号に規定されている
遊技機に関する型式の検定については、その有効期限が定められており、当該有効期限を
経過した後は、当該遊技機を営業所において遊技の用に供することができない。（123文字）
```

**適切な長さ（OK）**:
```
型式検定の有効期限が経過した遊技機は、営業所で使用することができない。（35文字）
```

---

### 4. 法的根拠の明記
**NG例**:
```
解説: 正解です。法律で定められています。
```

**OK例**:
```
解説: 正解です。風営法第20条第8項に規定されており、
型式検定の有効期限（3年）経過後の遊技機使用は禁止されています。
```

---

### 5. 数値の具体性
**NG例**:
```
営業許可の申請は一定期間前に行う必要がある。
```

**OK例**:
```
営業許可の申請は、営業開始予定日の30日前までに行う必要がある。
```

---

### 6. 否定形の適切な使用
**避けるべき例**:
```
遊技機の設置を行わない場合には届出を行わないことができない。
```
→ 二重否定で複雑

**改善例**:
```
遊技機の設置を行う場合には届出が必要である。
```

---

### 7. 絶対表現の注意
**NG例**:
```
遊技機の改造は絶対に禁止されている。
```
→ 「絶対」は例外がある場合に不適切

**OK例**:
```
遊技機の不正改造は禁止されている。
```
→ 「不正」と限定することで正確性を確保

## 📊 FP3級試験の問題文サンプル分析

### サンプル1: 基本的事実確認（易しい）
```
問題: 国民年金の第1号被保険者は、20歳以上60歳未満の自営業者等が該当する。
答え: ○
解説: 正しい。国民年金法第7条第1項に規定されています。
```

**特徴**:
- シンプルで明確
- 主語・述語が明確
- 法的根拠あり

---

### サンプル2: 数値の正誤（普通）
```
問題: 雇用保険の基本手当の受給要件は、離職日以前2年間に被保険者期間が
      通算して12ヶ月以上あることである。
答え: ○
解説: 正しい。雇用保険法第13条に規定されている原則的な受給要件です。
```

**特徴**:
- 期間・数値が具体的
- 条件が明確
- 例外がある場合は「原則的な」と注釈

---

### サンプル3: 手続き・タイミング（普通）
```
問題: 健康保険の被保険者資格を喪失した場合、資格喪失日から20日以内に
      保険証を返納しなければならない。
答え: ×
解説: 誤り。返納期限は資格喪失日から5日以内です。（健康保険法施行規則第50条）
```

**特徴**:
- 誤りの数値が「20日」→正しくは「5日」
- 1箇所だけの誤り
- 法的根拠が明確

## 🚀 遊技機主任者試験への適用指針

### 1. 難易度レベルの設定
**簿記3級・FP3級レベル**を基準とする：
- 専門用語は使用するが、複雑な法律解釈は避ける
- 基本的な条文知識の確認
- 実務で必要な知識の確認

### 2. 問題文の長さ
**目標**: 30-60文字
**最大**: 80文字まで
**最小**: 20文字（ただし説明不足に注意）

### 3. 解説の構成
```
解説: [正誤の判定]. [理由の説明].
法的根拠: [法令名]第○条第○項に規定されています。
補足: [実務的な注意点・関連知識]（任意）
```

### 4. カテゴリと法令の対応
**許容される法令**:
- 風営法（風俗営業等の規制及び業務の適正化等に関する法律）
- 風営法施行令
- 風営法施行規則
- 遊技機の認定及び型式の検定等に関する規則
- 各都道府県の風営法施行条例

**除外すべき法令**:
- 民法（契約一般）
- 商法（商行為一般）
- その他の一般法

### 5. 問題パターンの多様化
**参考にすべき試験**:
- FP3級: シンプルで明確な○×問題
- 宅建士: 実務的な知識の確認方法
- ビジネス実務法務: 簿記3級レベルの難易度調整

## 💡 重複問題を避けるための指針

### 1. セマンティックな重複の定義
以下の場合は「重複」とみなす：
- 問う内容が実質的に同じ
- 正誤を逆にしただけ（q0092 ⇔ q0106のような例）
- 言い回しが異なるだけで本質が同じ

### 2. 許容される類似パターン
以下の場合は「別問題」として許容：
- 同じ条文でも異なる論点を問う
- 同じテーマでも異なる数値・期間を問う
- 同じ手続きでも異なる段階を問う

**例**:
```
問題A: 型式検定の有効期限は3年である。（期限を問う）
問題B: 型式検定は国家公安委員会が行う。（実施機関を問う）
```
→ 同じ「型式検定」でも論点が異なるため許容
